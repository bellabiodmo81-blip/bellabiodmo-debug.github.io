📊 Contexte & Solution

❌ Le Problème

Les associations reçoivent des inscriptions de bénévoles par email ou formulaires papier. Elles les saisissent manuellement dans Excel, oublient de répondre, et perdent des bénévoles motivés.

  • Saisie manuelle longue
  • Pas de réponse immédiate
  • Données désorganisées
  • Perte de bénévoles

✅ La Solution

Un système automatisé de bout en bout qui capture les inscriptions, les enregistre dans une base de données, envoie une réponse immédiate, et notifie l'équipe. Zéro saisie manuelle.

  • Automatisation 100%
  • Réponse instantanée
  • Base de données centralisée
  • Notifications en temps réel

🛠️ Stack Technologique

Un système complètement No-Code, gratuit et scalable :

📋 Tally

Formulaire élégant et responsive pour l'inscription des bénévoles

🗄️ Airtable

Base de données avec vues Grille, Kanban et Calendrier

⚙️ Make

Orchestrateur d'automatisations via webhooks

📧 Gmail

Emails de bienvenue personnalisés

💬 Slack

Notifications équipe en temps réel

📱 Telegram

Alertes mobiles instantanées

🔄 Architecture & Flux

Le flux d'automatisation :

1

Inscription Tally — L'utilisateur remplit le formulaire avec ses infos (prénom, email, intérêts, disponibilité)

2

Webhook Make — Tally déclenche automatiquement Make via webhook

3

Création Airtable — Les données créent automatiquement un enregistrement dans la table "Bénévoles"

4

Email de bienvenue — Gmail envoie une réponse personnalisée en quelques secondes

5

Notifications équipe — Slack et Telegram alertent l'équipe en temps réel

📊 Dashboard Airtable

La base de données inclut 3 vues différentes pour piloter l'activité :

✓ Vue Grille

Affichage complet de tous les bénévoles avec leurs informations : prénom, nom, email, téléphone, ville, centres d'intérêt, disponibilité, statut, date d'inscription et notes.

✓ Vue Kanban

Groupement par statut (Nouveau → Contacté → Actif → Inactif). Permet de piloter visuellement l'avancement des inscriptions et d'identifier qui contacter.

✓ Vue Calendrier

Visualisation des inscriptions dans le temps, basée sur la date d'inscription. Utile pour identifier les périodes de forte activité.

3 Vues différentes
11+ Champs par bénévole
